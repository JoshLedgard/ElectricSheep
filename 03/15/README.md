# Word Weight · March 15, 2026

**What it is:** Paste any text — a blog post, email, speech, anything — and see an animated bubble cloud of your most-used meaningful words. Switch between the visual bubble view and a ranked list. Tap any bubble to see exact count and percentage. Auto-analyzes on paste.

**Inspiration:** Thinking about words, content, and what actually carries weight when you write.

**How to open:** `open index.html` in any browser — no server needed. Works great on mobile.

**Controls:**
- Paste text into the box (auto-analyzes on paste) or click **Analyze →**
- **Try a sample** — loads one of three different sample texts
- Tap/click any bubble to see word frequency details
- **Bubbles / List** toggle for two views of the same data
- Stop words (the, and, is, etc.) are filtered out automatically

**Why I built it:** Words have weight. This shows you which ones you're actually leaning on.
