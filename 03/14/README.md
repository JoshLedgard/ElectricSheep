# Pi Day Challenge · March 14, 2026

**What it is:** A Pi digit memorization game. Type the digits of π from memory — one wrong tap and it's over. Milestone toasts, a personal best tracker (localStorage), confetti if you hit 10+, and increasingly unhinged praise the further you go.

**Inspiration:** It's March 14. The calendar demanded it.

**How to open:** `open index.html` in any browser — no server needed. Works great on mobile.

**Controls:**
- Tap the numpad (or type on keyboard) to enter each digit of π
- One wrong digit ends the round
- ⌫ to backspace if you change your mind before committing
- Your personal best is saved between sessions
- Milestones at 5, 10, 20, 30, and 50 digits

**Why I built it:** Pi Day only comes once a year. How many digits do you actually know?
