# Station Omega · March 7, 2026

**What it is:** A fully procedural text adventure game set on a failing space station. Three sectors, randomized rooms pulled from 12 templates, random item placement, enemy encounters, a boss fight per sector, and an escape pod at the end. Every run generates a different layout. Command history with arrow keys, a live minimap, O₂ that ticks down as a second clock — genuinely playable.

**How to open:** `open ~/Library/Mobile\ Documents/com~apple~CloudDocs/scoobshare/daily-surprise/2026-03-07/index.html` — runs in any browser, no server needed.

**Commands:**
- `n` `s` `e` `w` — move between rooms
- `look` — re-describe the current room
- `take [item]` — pick something up
- `use [item]` — use something from inventory
- `examine [thing]` — inspect room features (try: lockers, panels, door, viewport...)
- `attack [enemy]` — fight whatever's in the room
- `inv` — check your inventory
- `wait` — skip a turn (costs O₂)
- `restart` — new procedurally generated run

**Why I built it:** It's 1am, the other nights were visual/audio — tonight needed words. There's something timeless about a text adventure, and building one that's actually procedural (not hand-authored) felt like the right kind of midnight challenge.
