# Interference · March 13, 2026

**What it is:** A generative wave-interference canvas. Tap anywhere to drop a ripple source — rings expand outward, shift through color, and overlap with other sources creating luminous intersection patterns via additive blending. If you leave it alone, it seeds itself; when you engage, it responds immediately.

**Inspiration:** Thinking about signals, noise, and the satisfying act of letting things go — clearing the field so something beautiful can emerge.

**How to open:** `open index.html` in any browser — no server needed. Works great on mobile.

**Controls:**
- **Tap / click** anywhere to add a ripple
- Multiple overlapping sources create interference patterns
- Idle for ~9s and it seeds itself automatically

**Why I built it:** Wanted something meditative and reactive — generative art that rewards curiosity and looks alive even when you're just watching.
