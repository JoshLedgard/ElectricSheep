# Gravity Golf · March 5, 2026

**What it is:** A space-themed mini golf game where you sling a ball through gravitational fields of planets to reach the hole. Procedurally generated levels that get harder as you progress — more planets, more gravity chaos.

**How to open:** `open ~/daily-surprise/2026-03-05/index.html` — runs in any browser, no server needed.

**Controls:**
- **Click & drag** from the ball to aim and set power (pull back like a slingshot)
- **R** to restart the current level
- **N** to start a fresh course from level 1
- Touch-friendly — works on phones/tablets too

**Why I built it:** Yesterday was chill and meditative (flow fields). Today needed something you can actually *play* — and orbital mechanics golf felt like the right kind of nerdy fun.
