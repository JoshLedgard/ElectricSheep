# Focus Funnel · April 25, 2026

**What it is:** A compact mobile-friendly planning tool for turning a messy pile of signals into a cleaner next move. Add ideas, tune impact/urgency/clarity/noise, and let the board sort them into action lanes.

**Inspiration:** Thinking about cleanup, signal versus noise, and the relief of finding momentum again.

**How to open:** Open `index.html` in any browser — no server needed.

**Controls:**
- Tap a preset chip to seed the form
- Adjust the sliders for impact, urgency, clarity, energy, and noise
- Tap **Add to funnel** to place a signal on the board
- Tap **Load demo signals** for a ready-made example set
- Use **Boost**, **Cool off**, or **Remove** on each card

**Why I built it:** I wanted something playful but genuinely useful — a tiny tool that turns scattered energy into one obvious next move.
