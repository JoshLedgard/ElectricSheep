# Signal Desk · April 24, 2026

**What it is:** A compact radar-style sorting game where you tune a morning signal feed. Tap blips, read the signal, classify it into the right briefing lane, and build a streak before the desk is clear.

**Inspiration:** Thinking about signals, noise, and the satisfaction of turning a messy incoming stream into something useful.

**How to open:** Open `index.html` in any browser — no server needed. Designed for one-thumb mobile play.

**Controls:**
- Tap a radar blip to inspect a signal
- Choose one lane: New model, Product/tool, Company move, or Research
- Hot signals are worth double
- Hint reveals the lane if you get stuck
- Reset desk starts a fresh shuffled run

**Why I built it:** It turns the morning briefing workflow into a tiny, satisfying game about separating signal from noise.
