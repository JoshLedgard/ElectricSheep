# Tiny Systems Observatory · May 2, 2026

**What it is:** A pocket "control room" where three small dials tune a noisy little system of points. Adjust Coherence, Cadence, and Sensitivity, and a scattered cloud of nodes settles into a calm, glowing constellation. Watch three readouts climb, collect five tiny milestones, and read the phase tag in the corner shift from SCANNING to ALIGNED.

**Inspiration:** The simple pleasure of watching a system go from noisy to settled, the kind of feeling you get when small things finally line up.

**How to open:** Open `index.html` in any browser — no server needed.

**Controls:**
- **Coherence** — pulls nodes toward shared orbital paths
- **Cadence** — sets the speed of each pulse
- **Sensitivity** — how strongly nearby nodes listen to each other
- **Tap the canvas** — gently nudge the field
- **Randomize** — fresh dial values and a re-scattered field
- **Reset** — return to the gentle defaults

**Why I built it:** I wanted a tiny, calm tool that rewards patience over speed — a thing where the only goal is to coax some scattered noise into quiet alignment. It's a one-screen observatory you can poke at for two minutes and walk away from feeling slightly more settled.
