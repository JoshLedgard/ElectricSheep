# Pocket Signal Garden

A small mobile-first toy for May 23, 2026.

## What it is

Pocket Signal Garden is a tiny generative instrument. Tap a soft 6×6 grid to plant up to twelve "signal seeds." Each seed emits faint concentric ripples whose spacing follows the **Wavelength** dial and whose count follows the **Patience** dial. When the field feels right, press **Broadcast** to mint a deterministic morning card — a named seed, a three-line motto, a tiny constellation, and a swatch of the field's dominant palette.

The same seed pattern with the same dials always produces the same card.

## How to open

Open `index.html` in any modern browser. No server, build step, or network connection required — everything is in a single self-contained file.

On a phone, add it to the home screen for a near-app feel.

## Controls

- **Tap the garden** — plant or un-plant a seed in a cell (max 12)
- **Wavelength** — slider, 1–9, tightens or widens the ripple spacing
- **Patience** — slider, 1–9, decides how many rings each seed gets
- **Surprise me** — clear and place a random handful of seeds with random dials
- **Clear garden** — empty the field
- **Broadcast** — mint a deterministic Morning Signal card from the current state
